# ecommerce-profit-maximization-analysis

## 💡 プロジェクト概要

このプロジェクトは、提供されたEコマースデータ(`EcommerceVisualization.csv`)を用いて、企業の**収益最大化**を最終目標（KGI）としたデータ分析と具体的なビジネス提案を行います。

「課題定義 → 仮説 → 分析 → 提案」のKGI軸に沿って進行し、データクリーニングの過程から、現場で実践可能な「泥臭いアクション」とROI（投資対効果）を意識した具体的な提案までを重視しています。

## 📊 使用データ

-   `archive.zip` 内の `EcommerceVisualization.csv`

## 🛠️ データ準備と初期探索

1.  `archive.zip` を解凍し、`EcommerceVisualization.csv` を特定。
2.  初期読み込みでエンコーディングエラー(`'utf-8' codec can't decode byte 0xa3`)が発生したため、`shift_jis` エンコーディングで再読み込みを実施。
3.  以下の課題が特定されました:
    -   `InvoiceNo` 列の混合型警告 (`DtypeWarning`)
    -   `InvoiceDate` が `object` 型
    -   `CustomerID` に約25%の欠損値
    -   `Quantity` と `UnitPrice` に潜在的な不正値（負の値や0以下の値）

## 🧹 データクリーニングと前処理

収益最大化のための分析基盤を確立するため、以下のデータクリーニングと前処理を行いました。

-   **`InvoiceNo` の処理:** キャンセルを示す 'C' を除去し、`IsCancelled` フラグを追加。
-   **`InvoiceDate` の型変換:** `datetime`型への変換を試みましたが、データフォーマットの多様性により多数の`NaT`（欠損日付）が発生 (`dayfirst=True`を適用しても約57%が`NaT`）。時間ベースの分析では、`NaT`ではない行を対象としました。
-   **`CustomerID` の欠損値処理:** 顧客ベースの分析のため、`CustomerID`が欠損している行（131,326行）を削除。
-   **`Quantity` と `UnitPrice` の異常値処理:** `Quantity`が負の値、または`UnitPrice`が0以下の行は分析から除外（今回のデータでは該当する行は0件）。正の数量と単価を持つ有効な取引のみを対象。
-   **`TotalPrice` 列の追加:** `Quantity` × `UnitPrice` で各トランザクションの総売上を示す `TotalPrice` 列を新規追加。

## 📈 データ分析と結果

### 1. 全体的な売上トレンドの把握

-   `InvoiceDate`が有効なデータに限定し、月次総売上トレンドを分析しました。
-   **結果:** 月次売上は強い季節性を示し、12月にピークがあり、1月から3月にかけて売上が低迷する傾向が確認されました。グラフ表示のために`japanize_matplotlib`を導入しました。

### 2. 顧客行動の分析 (RFM分析)

Recency（最終購入日からの経過日数）、Frequency（購入頻度）、Monetary（購入金額）の3つの指標を用いて顧客をセグメント化しました。

-   `pd.qcut` における `ValueError`（Bin labels must be one fewer than the number of bin edges）が発生したため、`duplicates='drop'` を適用し、動的にカテゴリを生成して1ベースのスコアに変換する修正を行いました。
-   **結果:** 以下の4つの顧客セグメントが特定されました。
    -   **その他 (Others):** 1541人 (大部分を占める、多様な行動パターン)
    -   **離反危険顧客 (At Risk):** 1144人 (Recency/Frequencyが低い)
    -   **高価値新規顧客 (New High-Value Customers):** 236人 (Recency/Monetaryが高い)
    -   **有望顧客 (Promising):** 61人 (Recencyが高く、Frequencyも比較的高い)

### 3. 商品パフォーマンス分析

商品ごとの総売上を計算し、売上上位10商品（ベストセラー）と売上下位10商品（死に筋商品）を特定しました。

-   **結果:**
    -   **売上上位商品:** 「PAPER CRAFT , LITTLE BIRDIE」、「REGENCY CAKESTAND 3 TIER」などがビジネスの主要な収益源であることが判明。
    -   **売上下位商品:** 「PADS TO MATCH ALL CUSHIONS」（0.003 GBP）など、極めて売上が低い商品が多数存在。在庫コストや管理コストの面で非効率である可能性が示唆されました。

## 🚀 収益最大化のための提案

### RFM分析に基づくリテンション施策（離反危険顧客向け）

**目的:** 顧客との接点を再構築し、再購入を促進することで、顧客離反を防ぎ収益を維持・向上させる。

1.  **パーソナライズされた再エンゲージメントメール/プッシュ通知:**
    -   **泥臭いアクション:** 最終購入日から一定期間経過した顧客を自動リストアップし、過去の購入履歴に基づくおすすめ商品や限定割引クーポンを自動配信するシステムを構築。
    -   **ROI意識:** クーポン利用率、再購入率をKPIとし、費用対効果を測定。
2.  **アンケートを通じたニーズ把握:**
    -   **泥臭いアクション:** 離反顧客に簡単なアンケートを実施し、利用がない理由や不満点を収集。回答者には謝礼を提供。
    -   **ROI意識:** 回答率とサービス改善への寄与度（定性・定量）を評価。
3.  **限定的なコミュニティへの招待:**
    -   **泥臭いアクション:** 離反危険顧客の中からエンゲージメントが高まりそうな顧客を選定し、SNSグループやオンラインイベントへ招待。
    -   **ROI意識:** 参加率、コミュニティ内でのエンゲージメント、再購入率の増加を追跡。
4.  **カスタマーサポートからのパーソナルなフォロー:**
    -   **泥臭いアクション:** 高額購入履歴のある離反顧客に対し、カスタマーサポートチームが個別に連絡を取り、状況確認や課題ヒアリングを実施。
    -   **ROI意識:** フォロー後の再購入率、顧客満足度、顧客単価の維持を評価。

### 商品パフォーマンスに基づく利益最大化施策

**目的:** ベストセラー商品の売上をさらに拡大し、死に筋商品の非効率性を解消することで、全体的な利益率を向上させる。

#### ベストセラー商品への施策

1.  **関連商品のクロスセル・アップセル推奨:**
    -   **泥臭いアクション:** ECサイトにレコメンデーション機能を強化（「一緒に購入されることが多い商品」など）。メールマーケティングでパーソナライズ提案。
    -   **ROI意識:** レコメンデーションからの購入率、平均注文額（AOV）の向上。
2.  **限定バンドル販売・セット販売の強化:**
    -   **泥臭いアクション:** ベストセラーと関連商品をセットで提供。在庫過多の関連商品をバンドルに含め在庫消化も図る。
    -   **ROI意識:** バンドル販売によるセット購入率、AOV、在庫回転率の改善。
3.  **プレミアム版・限定版の展開:**
    -   **泥臭いアクション:** 顧客フィードバックやトレンドを分析し、高単価のプレミアム版や限定版を企画・製造。先行予約キャンペーンを実施。
    -   **ROI意識:** プレミアム版・限定版の販売個数、単価、全体売上・利益への貢献。

#### 死に筋商品への施策

1.  **大幅な価格改定・クリアランスセール:**
    -   **泥臭いアクション:** 定期的にクリアランスセールを開催。特設ページやメルマガで告知し、早期に在庫を消化。
    -   **ROI意識:** 在庫滞留期間短縮、倉庫費削減、現金回収。売却損益の最小化。
2.  **バンドル販売・おまけとしての活用:**
    -   **泥臭いアクション:** 人気商品購入時におまけとして死に筋商品を提供、またはバンドルに含める。「今だけ！〇〇ご購入で限定〇〇プレゼント！」キャンペーン。
    -   **ROI意識:** ベストセラー販売促進効果、顧客満足度向上、死に筋在庫削減効果。
3.  **商品情報の見直し・プロモーション強化:**
    -   **泥臭いアクション:** ECサイトの商品ページを改善（写真、説明文、利用シーン）。ターゲット層を見直した広告出稿やSNSでの紹介。
    -   **ROI意識:** ページ閲覧数、転換率（CVR）の変化。プロモーション費用対効果。
4.  **商品ラインナップからの廃止検討:**
    -   **泥臭いアクション:** 定期的な商品ポートフォリオ見直し会議で、売上、利益率、在庫、トレンドを総合評価し、廃止候補を選定。廃止にかかるコストを算出。
    -   **ROI意識:** 不採算商品廃止によるコスト削減、新規商品開発へのリソース再配分。
